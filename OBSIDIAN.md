# Conceito do Projeto

Criar um web app que permita com que o usuário crie vários automatons, criando states e events, que se correlacionam entre si através de um canva criado pelo próprio usuário. Permitir a criação de várias abas com estes automatons, e no final, uma execução de simulação real do automaton, exibindo em uma janela personalizada a execução, e também um log de eventos embaixo desta janela.

Inspiração: https://lacsed.github.io/ultrades-app-teste/ | https://madebyevan.com/fsm/

# Teorias Aplicadas
Discrete Event Systems
States
Events
Automatons
Deterministic Finite Automatons
